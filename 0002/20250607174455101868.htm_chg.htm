WebSite-Watcher - WARNINGS:

  • WebSite-Watcher is not able to detect updates with the current bookmark configuration, the checked page has no text content!
    Possible solutions:
    1. If a server sometimes sends an empty page, you can specifiy the minimum number or characters a page must have (Bookmark properties + Advanced + Ignore Updates)
    2. For pages that require Javascript you can enable the check technology Chromium, Edge or IE (Bookmark properties + Check tab)
    3. Use the checking method "Check by screenshot" for pages that contain only images.
